# Expense Tracker 🧾

"Expense Tracker"는 지출 기록을 추가, 수정, 삭제할 수 있는 간단한 가계부 앱입니다.

---

## **기술 스택**
프론트엔드:
- `React Native`
- `Expo`
- `Context API` (상태 관리)
- `Axios` (HTTP 요청)

백엔드:
- `Firebase Realtime Database`
